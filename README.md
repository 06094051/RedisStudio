# 介绍
1. Redis Studio 是Redis桌面客户端，支持Windows xp, windows 7, windows 8
2. 界面基于duilib开发，内核客户端采用 MSOPEN hiredis
3. 支持 官方Redis 2.6 2.7 2.8
